## 三、程序流程控制

### 1.  程序流程控制概述

三种基本流程控制结构

* 顺序结构

  程序从上往下逐行执行，中间没有任何判断和跳转

* 分支结构

  根据条件，选择性执行某段代码

  `if ... else` 和 `switch-case`两种分支语句

* 循环结构

  根据循环条件，重复性执行某段代码

  `while` 、`do ...while` 和 `for`三种循环语句

  JDK1.5提供了 `foreach`循环，方便遍历集合、数组元素
